Automatically create videos for youtube

## Run Locally
Clone the project

```
$ git clone https://github.com/anekobtw/yt-videos-maker
```

Go to the project directory

```
$ cd src
```

Install dependencies

```bash
$ pip install -r requirements.txt
```

Run the program

```bash
$ python main.py
```
